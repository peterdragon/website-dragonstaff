lUse globbing for matching and allow the "admin" group to have rights for all processes. There is no page restriction, so the rights are valid for all objects
l
l ! match: glob   
l group admin    
l process *
lThe chiefeditors have rights for the processes "release", "publish" and "edit". Here too, there are no page restrictions
l
lgroup chiefeditor
l process release publish edit
