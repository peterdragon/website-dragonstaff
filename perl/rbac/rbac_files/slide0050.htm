Perl Implementations of RBAC
lI know of no solutions in Perl although there are libraries for Python, Ruby, Java. In principle you could wrap one of them lWe needed one at the BBC so I wrote one called IFL::Authz and hope to release it to CPAN lBased on Ferraiolo et al. "Proposed NIST Standard for Role-Based Access Control" l This paper has a Functional Specification of an API written in the Z formal language which I adapted to Perl. Z is nice match for the mathematical set theory underlying RBAC though there are some errors in the paper.
