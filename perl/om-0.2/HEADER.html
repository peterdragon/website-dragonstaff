Sample OpenInteract Application RedHat Linux Install Script

This directory tree contains a sample install suite for an OpenInterfact application on RedHat Enterprise Linux 3 or 4 running perl 5.8.0.
See ominstall.pl for details.
There is a templated Apache control/configuration setup utility in src/oiapachectl.pl with SQL definition in src/sys.sql.
The sub-directories otherwise contain dummy archive files.
